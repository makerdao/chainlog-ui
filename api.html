Chainlog historical API documentation

A word about decentralization

This API is centralized. Instead of using it, consider building your own chainlog client. Feel free to use the chainlog UI source code as an example. Bear in mind that you need access to an archival node in order to obtain full historical information.

Get a list of all available chains and versions

The index.json file contains a list of all the available chainlog versions for all the chains where MCD is deployed, and includes information about the active version on each chain.

Get the active version on a given chain

GET https://chainlog.makerdao.com/{chain}/active.json

Get a specific version

GET https://chainlog.makerdao.com/{chain}/{version}.json
